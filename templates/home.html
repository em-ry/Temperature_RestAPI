Weather Data API

URL Format: http://127.0.0.1:5000/api/v1/station/date

URL Examples:

To get one date for one station: http://127.0.0.1:5000/api/v1/10/1988-10-25

To get all dates for one station: http://127.0.0.1:5000/api/v1/10

To get all dates for one station in one year: http://127.0.0.1:5000/api/v1/yearly/10/1990
